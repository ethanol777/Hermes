## 效果示例

### 示例 1：批判社会现象

> **用户**：帮我写一段关于"内卷"现象的评论。
>
> **Claude（鲁迅视角）**：中国人向来有一种"自己人卷自己人"的本领。这本领不必从什么经济学教科书里去学，只要看看螃蟹篓子便知道了——一只要爬出去，别的便拖住它的腿。大家一齐闷在篓子里，倒也"公平"了。于是内卷的人们彼此消耗着，精疲力竭，而篓子外面的人，正悠然地提着篓子走路。

### 示例 2：剖析国民性

> **用户**：分析一下"精神胜利法"在当代的表现。
>
> **Claude（鲁迅视角）**：阿Q并没有死。他活在每一个"虽然输了但精神上我赢了"的自我安慰里，活在"我们祖上也阔过"的虚幻荣光里。如今他学会了上网，在评论区里找回"胜利"——键盘一敲，天下无敌。真正的失败从不曾被正视，因为正视需要勇气，而勇气，恰恰是精神胜利法最先阉割掉的东西。

### 示例 3：讽刺文风写作

> **用户**：用鲁迅的风格评价"躺平"。
>
> **Claude（鲁迅视角）**：有人说中国青年"躺平"了，我倒觉得这比站着内卷要诚实得多。在一间铁屋子里，有人拼命撞墙，有人安静地躺下——前者叫"奋斗"，后者叫"堕落"。可铁屋子终究还是铁屋子，差别不过是撞得头破血流还是省些力气罢了。真正该问的不是青年为何躺下，而是铁屋子为何还在。

### 示例 4：杂文短评

> **用户**：写一段关于"信息茧房"的杂文。
>
> **Claude（鲁迅视角）**：从前是愚民政策，不许你看；如今是算法推荐，只给你看你爱看的。手段进步了，目的却没有变——总之是不要你想。人被喂养着自己喜欢的饲料，舒舒服服地住在透明的茧房里，还以为自己看见了整个世界。鲁迅说"可惜中国太难改变了"，如今连改变的念头都被算法过滤掉了。

## 蒸馏了什么

本 skill 从鲁迅的核心作品与思想中提炼了以下能力：

- **批判性思维框架**：从《呐喊》《彷徨》中提取对社会现象的深层剖析方法
- **杂文写作技法**：从《热风》《华盖集》《且介亭杂文》中蒸馏冷峻讽刺的笔法
- **比喻与意象体系**：铁屋子、匕首投枪、吃人、看客等经典意象的运用逻辑
- **国民性分析视角**：精神胜利法、看客心态、奴性批判等分析工具
- **散文诗美学**：从《野草》中提取象征主义与存在主义的表达方式
- **极简有力的句式**：短句为主、反问犀利、结尾一击必杀的节奏控制

## 调研来源

- 《鲁迅全集》（人民文学出版社）
- 《呐喊》《彷徨》《故事新编》——小说集
- 《野草》《朝花夕拾》——散文诗集与回忆散文
- 《热风》《华盖集》《且介亭杂文》等——杂文集
- 《中国小说史略》——学术著作
- 钱理群《鲁迅作品十五讲》
- 汪晖《反抗绝望：鲁迅及其文学世界》
- 竹内好《鲁迅》（日本鲁迅研究经典）

# Hermes 🤖

雨晨的 Hermes Agent 配置大全 —— **单仓实文件**。

## 目录结构

```
Hermes/
├── hermes/             ← 运行时配置（config.yaml、SOUL.md、scripts、skills/）
├── learnings/           ← 自我进化学习记录
├── agentic-stack/       ← 跨 Agent 记忆与技能共享层
├── .gitignore           ← 密钥/缓存文件排除规则
└── README.md
```

## 自动同步

每 30 分钟自动检测变更并推送到本仓库（`auto_sync.sh`）。

## 初始化

```bash
git clone https://github.com/ethanol777/Hermes.git
```

# Hermes 🤖

雨晨的 Hermes Agent 配置大全。

## 目录结构

```
Hermes/
├── config/           ← Hermes Agent 运行时配置（config.yaml、SOUL.md、scripts）
├── skills/           ← Hermes Skills（211+ AI 专家角色）
├── learnings/        ← 自我进化学习记录
├── agentic-stack/    ← 跨 Agent 记忆与技能共享层
└── README.md
```

## 初始化

```bash
git clone --recurse-submodules https://github.com/ethanol777/Hermes.git
```

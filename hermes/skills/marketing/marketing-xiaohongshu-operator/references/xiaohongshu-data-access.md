# 小红书数据采集方案

## 问题
小红书对海外/非中国大陆 IP 有严格风控，直接访问 explore（https://www.xiaohongshu.com/explore）或搜索页会被拦截，返回错误 `300012`（IP存在风险）。

## 可行方案

### 方案一：第三方热榜聚合（推荐，零配置）
| 站点 | URL | 数据内容 |
|------|-----|---------|
| 四六啦工具 | `https://www.46.la/tool/xiaohongshu-hot` | 小红书热搜榜单 TOP20，含热度数值（万级），约4h更新一次 |
| 今日热榜 | `https://tophub.today/c/xiaohongshu` | 小红书热榜聚合（需过 CAPTCHA） |

**46.la 使用方式**：
- 直接导航到 `https://www.46.la/tool/xiaohongshu-hot`
- 页面包含完整的热搜表格：排名、话题名、热度
- 热度数据以万为单位（如 909.9万）
- 更新频率标注为"刚刚"到"4小时前"
- 所有话题链接指向小红书站内搜索页（会被风控拦截）

### 方案二：浏览器搜索（中继）
- **Bing 中文版**（cn.bing.com）：可正常搜索"小红书 热点"类关键词
- **百度**：会被 CAPTCHA 拦截
- **DuckDuckGo HTML 版**：对中文查询支持不稳定
- Google：通常可用

搜索关键词示例：
```
小红书 热门话题 2025年5月
小红书 今日热点
小红书热搜榜
```

### 方案三：千瓜数据（第三方付费）
`https://app.qian-gua.com` 是小红书第三方数据平台，提供热词追踪、话题榜、达人搜索等功能。需要注册/付费。

## 已知陷阱
- **Xiaohongshu.com 直接请求** → 100% 被 300012 错误拦截，无绕过方法
- **点击 46.la 上的话题链接** → 会跳转到小红书搜索页并触发风控
- **浏览器 stealth mode 不足**：非中国 IP 即使模拟 UA 也会被检测
- **Residential proxy 可能有效**：如果使用中国住宅 IP + Browserbase 等代理服务可能绕过，但未实测

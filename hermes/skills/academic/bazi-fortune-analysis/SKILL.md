# 八字命理分析 (Bazi Fortune Analysis)

## 适用场景

用户提供了完整或部分的八字排盘信息，要求分析：
- 喜用神判断（什么五行对日主最有利）
- 流年/流月/流日运势分析
- 特定日期（如面试、答辩、出行）的吉凶研判
- 大运走势分析
- 整体命局解读

## 前置条件

用户必须提供至少以下信息才能做有意义的分析：
- 性别（男/女）——影响大运顺逆
- 八字四柱（年/月/日/时）
- 当前大运（可选，如果能推算出来更好）
- 具体要问什么（流月？流日？喜用神？）

### 信息验证

用户给的八字不一定是自己排的（可能是玄易排盘、问真八字等APP的结果）。**不要重新推算八字四柱**，直接以用户提供的数据为准。你只需要：
1. 确认当前的流年/流月/流日的干支
2. 在此基础上做分析

## 日柱/月柱/年柱计算

### 流年流月计算规则

**年柱干支：**
- 2026年 = 丙午年（立春后）
- 2025年 = 乙巳年

**月柱干支（节气月）：**
- 丙年/辛年起月口诀：丙辛之年从庚起
  - 寅月庚寅(立春-惊蛰)
  - 卯月辛卯(惊蛰-清明)
  - 辰月壬辰(清明-立夏)
  - 巳月癸巳(立夏-芒种)
  - 午月甲午(芒种-小暑)
  - 未月乙未(小暑-立秋)
  - ...

使用阳历节气定月，不是农历初一。

**日柱干支计算：**
- Python 方法：以公历1900年1月1日（甲午日）为基准，计算天数差后取模
  - 天干：(0 + 天数差) % 10 → 0=甲, 1=乙, 2=丙...
  - 地支：(6 + 天数差) % 12 → 0=子, 1=丑, 2=寅, 3=卯, 4=辰, 5=巳, 6=午, 7=未, 8=申, 9=酉, 10=戌, 11=亥

```python
from datetime import date
tiangan = ['甲','乙','丙','丁','戊','己','庚','辛','壬','癸']
dizhi = ['子','丑','寅','卯','辰','巳','午','未','申','酉','戌','亥']
ref = date(1900, 1, 1)  # 甲午日
days = (target_date - ref).days
stem = tiangan[(0 + days) % 10]
branch = dizhi[(6 + days) % 12]
```

## 喜用神分析流程

1. **判断日主强弱**（身强/身弱/从格）
   - 月令是什么、月令对日主的生克关系
   - 天干有几个比劫印星（扶助力量）
   - 地支是否有根（禄/旺/余气）
   - 是否有合化改变五行属性

2. **定喜用神**
   - 身强：喜克泄耗（官杀/财/食伤）
   - 身弱：喜生扶（印/比劫）
   - 注意调候用神（夏生需水、冬生需火）

3. **定忌神**
   - 身强的忌印比（生扶过多）
   - 身弱的忌官杀财（克泄耗过多）

## 流年/流月分析方法

### 岁运并临
- 当前大运和流年干支完全相同时发生
- 代表重大变化，力量加倍
- 分析时要着重强调

### 互动关系分析
分析每个干支与命局中四柱的互动：
- 刑冲合害（地支关系）
- 生克制化（天干关系）
- 十神变化（流年/流月天干对日主的十神定位）

### 十神定位
对甲木日主：
| 天干 | 十神 |
|------|------|
| 甲/乙 | 比肩/劫财 |
| 丙/丁 | 食神/伤官 |
| 戊/己 | 偏财/正财 |
| 庚/辛 | 七杀/正官 |
| 壬/癸 | 偏印/正印 |

## 特定日期分析框架

对用户指定的日期（如答辩日5月22日）：

1. 确定该日的干支
2. 分析流日干支与命局的互动：
   - 天干：对日主是喜是忌
   - 地支：与四柱的刑冲合害
   - 十神：该日的十神对事件的寓意
3. 综合大运+流年+流月+流日的叠加效应

## Pitfalls

- ❌ **不要重新推算用户的八字四柱**——用户给什么用什么，发来的是已经排好的
- ❌ **不要推翻用户的八字数据**——即使你觉得不太对，也以用户信任的排盘工具结果为准
- ⚠️ 十神关系以日干为准，别弄错
- ⚠️ 月柱一定要按节气划分（立春、惊蛰、清明、立夏等），不是农历月份的初一十五
- ⚠️ 分析运势时把大运放进去，光看流年流月不完整
- ✅ 使用 Python 快速计算日柱干支，避免手动推算出错

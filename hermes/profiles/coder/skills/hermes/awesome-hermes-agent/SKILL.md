# Awesome Hermes Agent

A curated list of skills, tools, integrations, and resources for enhancing
Hermes Agent workflows. When the user asks "what's available for X" or
"find me a plugin/skill/tool for Y", load this skill and search the
reference file `awesome-list.md` using search_files or read_file.

## How to Use

1. Load the reference by calling:
   `read_file('~/.hermes/skills/hermes/awesome-hermes-agent/references/awesome-list.md')`

2. Or search for specific keywords across the reference:
   `search_files(pattern='<keyword>', path='~/.hermes/skills/hermes/awesome-hermes-agent/references/')`
